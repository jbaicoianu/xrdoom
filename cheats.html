Cheats
HTML Doom supports many of the same cheat codes that the original Doom supported. Here's a list of useful ones.

idclev##
   Change levels. Replace ## with the episode and level number - for example, idclev15 would change to episode 1, level 5. Remember though, the shareware version of Doom and Heretic only have episode 1!

idspispopd
   Toggles noclipping mode so you can walk through walls

idclip
   Same as idspispopd
